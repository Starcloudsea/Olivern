# Olivern
一个优化Mod, 旨在提升你在游戏内的帧数.

## 有什么作用?
- 大幅提升你的帧数(在验证后30FPS可以提升至300FPS.)
- 完全兼容VulkanMod(兼容OpenGL光影)
- 修复Minecraft一些未被发现的潜在内存泄漏.
- 降低Minecraft的性能占用(你可以再额外开一些应用.)
